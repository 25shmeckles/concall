medaka.util.pileup.bam_to_feature_array(reads_bam, ref, start=None, end=None)[source]¶

Converts a section of an alignment pileup (as shown by e.g. samtools tview) to a base frequency feature array

Parameters:
  • reads_bam – (sorted indexed) bam with read alignment to reference
  • ref – name of reference to process
  • start – starting position within reference
  • end – ending position within reference
Returns:

tuple(positions, feature_array)

  • positions: numpy structured array with ‘ref_major’ (reference position index) and ‘ref_minor’ (trailing insertion index) fields.

  • feature_array: 2D numpy array where each row is an 11-element feature array for a pileup position: array([(encoded) reference base, nA+, nC+, nG+, nT+, nDel+, nA-, nC-, nG-, nT-, nDel-])

    (e.g. nA+ = num. forward reads aligning with A to ref pos) (reference bases are encoded by feature_index())
Example: 
Outputs for a minimal pileup with 3 aligned reads:

reference sequence: AC**T
             read1: ACG*T  (forward alignment)
             read2: ACGGT  (forward alignment)
             read3: ac**t  (reverse alignment)

positions:

    ref_major  ref_minor
    1          0
    2          0
    2          1
    2          2
    3          0

feature_array:

    array([[1, 2, 0, 0, 0, 0, 1, 0, 0, 0, 0]   (2xA+, 1xA-)
           [2  0, 2, 0, 0, 0, 0, 1, 0, 0, 0]   (2xC+, 1xC-)
           [5  0, 0, 2, 0, 0, 0, 0, 0, 0, 1]   (2xG+, 1xDel-)
           [5  0, 0, 1, 0, 1, 0, 0, 0, 0, 1]   (1xG+, 1xDel+, 1xDel-)
           [4  0, 0, 0, 2, 0, 0, 0, 0, 1, 0]]) (1xT+, 1xT-)
